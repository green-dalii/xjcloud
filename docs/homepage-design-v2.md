# 首页重构设计方案 v2

日期：2026-05-22
来源：用户提供的详细交互设计

---

## 核心策略：「轻触即达，趣味留白」

### 一、双轨制入口

不采用传统"选择身份"弹窗，而是**"一屏双生"**：

```
[全屏手绘地图风格背景]
        ↓
左下角悬浮按钮          右下角悬浮按钮
"我要探索" (游客)        "我要造趣" (创建者)
   ↓                      ↓
  脉冲动效               笔触感效
```

**为什么：**
- 地图即内容——乡村文旅的核心是地理探索
- 不切割用户——20-40岁人群可能今天游客明天创建者
- 手绘风格建立"乡村松弛感"第一印象

### 二、色彩体系（更新）

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色-泥土棕 | `#8B6F47` | 大地感 |
| 主色-稻田绿 | `#7A9E7E` | 自然感 |
| 强调色-落日橙 | `#E07A5F` | 关键按钮 |
| 背景-米纸白 | `#F7F5F0` | 避免纯黑白 |

### 三、20-40岁人群交互细节

| 痛点 | 解决方案 |
|------|---------|
| 讨厌注册 | 微信一键登录为主，首次进入直接看内容 |
| 审美疲劳 | 不用轮播Banner，用"呼吸式"卡片 |
| 选择困难 | 首页不堆活动列表，用"今日灵感" |
| 不信任感 | 创建者入口增加"看看别人怎么玩"案例 |

### 四、反直觉设计

**首屏不放搜索框。**

- 首屏：视觉地图 + 情绪标签（"想发呆""想动手""想社交"）
- 次屏：滑动浏览最近3天的活动
- 搜索框收在顶部，需要时上滑出现

### 五、入口后路径

```
游客路径：
地图探索 → 筛选(周末/亲子/露营/农耕) → 活动详情 → 一键报名 → 行前指南

创建者路径：
灵感库(看别人案例) → 快速创建(3步：地点/时间/玩法) → 智能定价建议 → 发布
```

### 六、微交互亮点（MVP 简化版）

1. **下拉刷新**：简化版——叶子飘落动画
2. **收藏活动**：点击爱心后飘出叶子
3. **创建完成**：屏幕轻微震动 + 电子印章效果
